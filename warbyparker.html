Warby Parker
Funnel Project

The project is from Codecademy's Analyze Data with SQL course. Project description: Warby Parker is a transformative lifestyle brand with a lofty objective: to offer designer eyewear at a revolutionary price while leading the way for socially conscious businesses. Founded in 2010 and named after two characters in an early Jack Kerouac journal, Warby Parker believes in creative thinking, smart design, and doing good in the world. For every pair of eyeglasses and sunglasses sold, a pair is distributed to someone in need.

In this project, I will analyze different Warby Parkerâ€™s marketing funnels in order to calculate conversion rates.


First things first, I want to get a feel for the data. There are three tables, I'll run three queries to see the first five rows of each dataset.

SELECT * FROM quiz LIMIT 5;


SELECT * FROM home_try_on LIMIT 5;


SELECT * FROM purchase LIMIT 5;



Next, I'm going to tie together the data using joins based on user_id. This will start to give us some valuable data.

 SELECT DISTINCT q.user_id,
  h.user_id IS NOT NULL AS 'is_home_try_on',
  h.number_of_pairs,
  p.user_id IS NOT NULL AS 'is_purchase'
 FROM quiz q
 LEFT JOIN home_try_on h
  ON q.user_id = h.user_id
  LEFT JOIN purchase p
 ON p.user_id = q.user_id;


Query Results


Let's start looking at conversion. Does everyone that takes the quiz go on to try on? What about purchasing?

 WITH q AS (
  SELECT '1-Quiz' as stage, COUNT(DISTINCT user_id) FROM quiz),
 h AS (
 SELECT '2-Home-Try-On' as stage, COUNT(DISTINCT user_id) FROM home_try_on),
 p AS (
 SELECT '3-Purchase' as stage, COUNT(DISTINCT user_id) FROM purchase)

 SELECT * FROM q
 UNION ALL
 SELECT * FROM h
 UNION ALL
 SELECT * FROM p;


Query Results



Now we a starting point for our conversion rate. Does conversion rate differ depending on if the user tried on 3 pairs or 5 pairs of glasses? How about for men's styles? Let's find out!

 WITH men AS (
  SELECT * FROM quiz
  WHERE style = "Men's Styles"),

  details AS (SELECT DISTINCT q.user_id,
  h.user_id IS NOT NULL as 'home_try_on',
  h.number_of_pairs as 'AB_variant',
  p.user_id IS NOT NULL as 'purchase'
  FROM men q
  LEFT JOIN home_try_on h
   ON q.user_id = h.user_id
  LEFT JOIN purchase p
   ON q.user_id = p.user_id),

  temp_details AS (SELECT AB_variant,
   SUM(CASE WHEN home_try_on = 1
   THEN 1
   ELSE 0
   END) as 'home_try_on_total',
   SUM(CASE WHEN purchase = 1
   THEN 1
   ELSE 0
   END) as 'purchase_total'
  FROM details
  GROUP BY AB_variant
  HAVING 'home_try_on_total' > 0)

 SELECT *, ROUND((1.0 * purchase_total / home_try_on_total),3) as 'conversion_rate'
 FROM temp_details
 WHERE home_try_on_total > 0;

Query Results


Analysis / Summary




The results of the last query clearly shows that men who try on five pairs of glasses are more likely to purchase than those who only try on three. The conversation rate is just over 90%. Could this be because the purchaser had more options and found a pair they really like? Before jumping to a conclusion it would be helpful to see the results for women and the entire population.
